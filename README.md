# SSG_crosswalking_api
This notebook contains codes to launch a crosswalking fastapi endpoint. To begin, please follow the below instructions.

1) Please git clone the remote repository onto your desktop
2) Install all package requirements from requirements.txt by doing `pip install -r requirements.txt`
3) Then please ensure you have docker cli installed on your terminal.
4) To build the image, run `docker build -t skill-matcher .`
5) To run the image, ca run `docker run -p 8000:8000 skill-matcher`
6) Then test the api end point on `http://0.0.0.0:8000/docs`
